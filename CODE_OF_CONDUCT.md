# Code of Conduct

## Our commitment

We want this project to be welcoming to contributors of every age, experience level, identity, location, and background. Students and first-time open-source contributors are especially welcome.

## Expected behavior

- Be respectful, patient, and constructive.
- Explain technical feedback clearly and focus on the work.
- Respect privacy and never post another person's credentials or customer data.
- Accept correction and help others learn.
- Avoid harassment, discrimination, threats, sexualized content, and personal attacks.

## Scope and enforcement

This code applies in repository issues, pull requests, discussions, and other official project spaces. Maintainers may edit or remove content, close contributions, or restrict participation when behavior is unsafe or repeatedly disruptive.

Report conduct concerns privately to the repository maintainer using an appropriate private GitHub contact method. Do not include secrets or unnecessary personal information. Reports will be reviewed as confidentially as practical, but this volunteer project cannot promise a specific response time.
