# Roadmap

This roadmap describes planned work, not completed features. Priorities may change after community feedback and security review.

## Near term

- Expand the reviewed Hinglish intent and FAQ test dataset.
- Add an authentication and rate-limiting deployment guide.
- Improve profile schema documentation and migration guidance.
- Add privacy-preserving, opt-in conversation storage design.

## Channel and business integrations

- Define and test a generic inbound/outbound channel adapter interface.
- Implement a Meta WhatsApp Cloud API adapter with signature verification and mocks.
- Add a booking-provider interface before any provider-specific implementation.
- Add CRM and SMS adapters only with sandbox tests and honest capability labels.

## Quality and community

- Add end-to-end deployment examples for one supported host.
- Grow a consent-safe Hinglish evaluation dataset using fictional prompts.
- Document maintainer release and vulnerability-response playbooks.
- Review accessibility and localization of future web examples.

## Explicitly out of scope for the current MVP

- Claiming production readiness.
- Storing real customer conversations by default.
- Shipping credentials or provider-specific paid accounts.
- Automatic marketing or unsolicited messaging.
