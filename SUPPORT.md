# Support

This is a volunteer-maintained alpha project.

## Before asking

1. Read the README and current limitations.
2. Run `local-biz-ai validate-config`.
3. Reproduce in `LOCAL_BIZ_PROVIDER=local` mode.
4. Run `python -m pytest` if you are developing locally.
5. Remove all credentials and personal data from logs or screenshots.

## Where to ask

- Use a bug report for reproducible project defects.
- Use a feature request for a scoped public open-source enhancement.
- Use a discussion, if enabled, for general questions.
- Follow SECURITY.md for vulnerabilities.

Include operating system, Python version, installation command, sanitized error text, expected behavior, and actual behavior. Support for private InceptionAIStudios client work is outside this public repository.
